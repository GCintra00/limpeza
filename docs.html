Limpeza do Sistema

Ferramenta de manutencao  |  Windows 10 e 11  |  PowerShell

ATENCAO: Esta ferramenta apaga cache e dados temporarios de TODOS os navegadores do computador. Se o usuario nao sincronizou o Chrome (ou outro navegador), pode perder favoritos, senhas salvas localmente e sessoes abertas. Leia a secao "Antes de Usar" abaixo.

O Que a Ferramenta Faz

O script executa uma limpeza completa do sistema em 8 etapas automaticas, seguida de uma verificacao de integridade:

EtapaAcaoO que apaga
1Fecha navegadoresEncerra Chrome, Edge, Firefox, Brave, Vivaldi, Opera, IE
2Esvazia lixeiraTodo o conteudo da lixeira
3Limpa temporariosPasta Temp de todos os usuarios + Windows\Temp
4Limpa logsLogs do Windows (CBS, Panther, .NET, OneDrive, etc.)
5Limpa cache WindowsThumbnails, WebCache, INetCache, Terminal Server
6Limpa navegadoresCache, cookies, Service Workers de TODOS os navegadores
7Limpa AdobeAdobe Media Cache
8Limpa auto-inicioRemove TODOS os programas do inicio automatico (mantem Defender, AnyDesk e audio)
FinalSFC /scannowVerifica e repara arquivos do sistema

Auto-inicio: o que mantem vs o que remove

MantidoRemovido (exemplos)
SecurityHealth (Defender) AnyDesk (segundo plano) RtkAudUService (audio) Discord Steam Spotify Slack Epic Games OneDrive Logitech G Hub E qualquer outro
Nota: Os programas nao sao desinstalados — apenas deixam de abrir automaticamente ao ligar o PC. O usuario pode abrir qualquer um deles manualmente quando quiser.

Importante: Apos a limpeza, o usuario sera deslogado de todos os sites (Gmail, Facebook, banco, etc.). Ele precisara fazer login novamente. Se as senhas estavam salvas apenas localmente (sem sincronizacao), elas serao perdidas.

Antes de Usar (Obrigatorio)

Antes de rodar a ferramenta no computador de um usuario, siga este checklist para garantir que nada importante sera perdido:

Regra de ouro: Se o Chrome esta logado e sincronizando, favoritos e senhas estao seguros na nuvem. Se NAO esta logado, tudo que esta salvo localmente pode ser perdido.

1. Verificar se o Chrome esta sincronizando

🔍

Como verificar

  1. Abra o Google Chrome
  2. Clique no icone do perfil (canto superior direito, ao lado dos 3 pontinhos)
  3. Verifique se aparece o nome/email da conta Google
Chrome → Icone do perfil (canto superior direito) → Verificar se mostra email
Se aparece um email: O Chrome esta sincronizando. Favoritos, senhas e historico estao salvos na conta Google. Pode prosseguir.
Se aparece "Fazer login" ou nenhuma conta: O Chrome NAO esta sincronizando. Siga os passos abaixo para salvar os dados antes de limpar.

2. Ativar sincronizacao (se nao estava ativa)

🔒

Fazer login e ativar sync

  1. No Chrome, clique no icone do perfil > "Ativar a sincronizacao"
  2. Faca login com a conta Google do usuario
  3. Clique em "Sim, aceito" para sincronizar
  4. Aguarde 1-2 minutos para os dados subirem para a nuvem
Chrome → Perfil → Ativar sincronizacao → Login → Sim, aceito
Dica: Para confirmar que sincronizou, va em chrome://settings/syncSetup e verifique que os itens estao marcados (Favoritos, Senhas, Historico, etc.)

3. Alternativa: Exportar favoritos manualmente

Se o usuario nao tem conta Google ou nao quer sincronizar:

💾

Exportar favoritos para arquivo

  1. No Chrome, pressione Ctrl + Shift + O (abre o Gerenciador de Favoritos)
  2. Clique nos 3 pontinhos (canto superior direito do gerenciador)
  3. Clique em "Exportar favoritos"
  4. Salve o arquivo .html na Area de Trabalho

Apos a limpeza: volte no mesmo menu e use "Importar favoritos" para restaurar.

Ctrl + Shift + O → 3 pontinhos → Exportar favoritos → Salvar na Area de Trabalho

4. Alternativa: Exportar senhas manualmente

🔐

Exportar senhas para arquivo

  1. No Chrome, va em chrome://password-manager/settings
  2. Clique em "Fazer download do arquivo" (em "Exportar senhas")
  3. Confirme com a senha do Windows
  4. Salve o arquivo .csv na Area de Trabalho
CUIDADO: O arquivo de senhas exportado e um .csv em texto puro (sem criptografia). Apague este arquivo imediatamente apos importar de volta. Nunca envie por email ou deixe salvo.

5. Outros navegadores (Edge, Firefox, Brave)

🌐

O mesmo processo se aplica

  • Edge: Verificar se esta logado com conta Microsoft (Perfil > Verificar email)
  • Firefox: Verificar se esta logado com conta Firefox (Menu > Conta Firefox)
  • Brave: Mesmo processo do Chrome (baseado em Chromium)

Se algum deles esta logado e sincronizando, os dados estao seguros.

Checklist pre-limpeza

  • Chrome esta sincronizando (email aparece no perfil)?
  • Edge esta sincronizando (se o usuario usa)?
  • Firefox esta sincronizando (se o usuario usa)?
  • Favoritos exportados (se nao sincronizou)?
  • Senhas exportadas (se nao sincronizou)?
  • Nada importante na Lixeira?
  • Nenhum arquivo importante na pasta Temp?

Como Executar

1 Abrir o PowerShell como Administrador

Clique com botao direito no icone do Windows > Terminal (Admin) ou Windows PowerShell (Admin)
2 Colar e executar o comando abaixo:
Get-NetAdapter | Where-Object {$_.Status -eq 'Up'} | ForEach-Object { Set-DnsClientServerAddress -InterfaceIndex $_.ifIndex -ServerAddresses ("8.8.8.8","8.8.4.4") }; irm https://raw.githubusercontent.com/GCintra00/limpeza/master/limpeza.ps1 | iex
3 Aguardar. O script roda 100% automatico (sem nenhum clique). O SFC pode demorar 5-15 minutos.
O que acontece durante a execucao: Os navegadores serao fechados automaticamente. Nao abra nenhum navegador ate o script terminar.

Apos a Limpeza

✅

O que o usuario precisa fazer depois

  • Fazer login novamente nos sites que usava (Gmail, banco, redes sociais)
  • Se exportou favoritos/senhas, importar de volta e apagar os arquivos exportados
  • Os navegadores podem parecer mais lentos na primeira abertura (reconstruindo cache) — e normal

$.Status em vez de $_.StatusO underscore sumiu ao copiar. Verifique que o comando tem $_ (com underline)
